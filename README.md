# AI Chess

An AI-powered Chess application built in Python that allows users to play against an intelligent computer opponent. The project implements the Alpha-Beta Pruning algorithm to efficiently evaluate moves while following official chess rules, including castling, legal move validation, check, checkmate, and stalemate detection.

---

## Features

- AI opponent powered by Alpha-Beta Pruning
- Complete legal move validation
- Kingside and Queenside castling
- Check and checkmate detection
- Stalemate and draw detection
- Drag-and-drop piece movement
- Legal move highlighting
- Reset game functionality
- Multiple AI difficulty levels
- Interactive graphical user interface

## Installation

Clone the repository:

```bash
git clone https://github.com/Athreya07/AI-Chess.git
```

Navigate to the project directory:

```bash
cd AI-Chess
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

---

## Gameplay

- Play against an AI opponent.
- Move pieces using drag-and-drop controls.
- The AI automatically calculates and executes its moves.
- The game enforces official chess rules, including castling and legal move validation.
- The game detects check, checkmate, stalemate, and draw conditions.
